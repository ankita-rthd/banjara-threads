# The Living Threads of Banjara

A small two-page static website about **Banjara embroidery** — an independent art form carried by the semi-nomadic Banjara (Lamani / Lambani) community. Made as a school project by **Ankita**.

**Live:** https://ankita-rthd.github.io/banjara-threads/

## What's here

- `index.html` — landing page with the hero, opening paragraph, a 4-tile mosaic of motifs, the artisans' shelf, a pull-quote, and a CTA.
- `art.html` — the long reader: origin & people, the fourteen traditional stitches, motifs and symbolism, the present day, and sources.
- `styles.css` — shared editorial design system (Fraunces + Cormorant Garamond, Banjara-inspired palette).
- `script.js` — sticky nav, smooth scroll, reveal-on-scroll.
- `assets/placeholders/` — hand-authored SVG illustrations used throughout the site.
- `tasks/` — the PRD and task breakdown the site was built from.

## Running locally

It's a plain static site — just open `index.html` in a browser. No build step, no dependencies.
